About the App

Eli's Pizza Picker helps groups decide on pizza orders through a simple voting system. Everyone ranks their top pizza choices, and the app calculates the optimal order based on everyone's preferences.

Features:

  • Easy voting with drag-and-drop ranking
  • Real-time vote updates
  • Smart order recommendations
  • Find nearby pizza places
